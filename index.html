💡 What is this tool?

This program acts as an assistant for your computer terminal. It uses artificial intelligence to help you write code, manage files, and execute tasks directly from your command line. It works with many different intelligence models, giving you flexibility in how you build software.

Because it runs as a single file on your computer, you keep full control over your data and your development environment.

📋 Requirements

  • Windows 10 or Windows 11.
  • A stable internet connection to talk to AI providers.
  • Basic knowledge of your Windows Terminal or Command Prompt.
  • At least 100MB of free disk space.

🚀 How to install and run

Follow these steps to set up the agent on your machine.

1. Visit the download page

Click the link below to visit the official release page. You will find a list of files for different systems.

2. Save the file

Look for the file that ends in .exe. Save this file to a folder you can easily find, such as your Downloads folder.

3. Run the application

Open your Windows Terminal or Command Prompt. Navigate to the folder where you saved the file. You can do this by typing "cd Downloads" and pressing Enter.

Type the name of the file followed by the word help to see a list of commands. This ensures the program is ready to work.

🔧 How to use it

Once you verify the installation, you can connect the tool to an AI provider. Many providers offer free tiers or low costs for developers. You will need an API key from your chosen provider. Provide this key to the agent when prompted during your first session.

The agent understands plain instructions. You can type requests like "Create a Python script that organizes my files" or "Refactor this function to be more efficient." The tool processes your request and provides the output directly in your terminal window.
